# Snapshot Data

These files make the screen tab load instantly on Streamlit Cloud without any
network calls.

## Files

| File | Rows | Source |
|---|---|---|
| `ia_funds.csv` | ~1762 | Cleaned IA fund list with triaged ISIN resolutions |
| `ia_screen_results.parquet` | ~1451 | Pre-computed alpha-screen results |

## Provenance

Generated by `scripts/build_snapshot.py` from the source repo's reports:

- **Funds:** `reports/ia_resolutions_20260708_triaged.csv` (Investment Association fund list + Morningstar ISIN resolution + manual triage review)
- **Screen:** `reports/ia_alpha_screen_checkpoint_matched_500_w4.csv` (alpha analysis output for matched funds)

Generated: 2026-07-17.

## Contents

Public data only: fund names, management company names (both public), ISINs,
IA sector classifications, and derived alpha metrics. No personal data,
no portfolio holdings of individuals.

## Refreshing

After updating the source data in the private repo:

```bash
python scripts/build_snapshot.py \
    --funds <path-to-triaged-csv> \
    --screen <path-to-checkpoint-csv> \
    --out data
git add data/
git commit -m "Refresh IA snapshot"
```

## Live data

The screen tab's "Re-run live" button bypasses this snapshot and fetches fresh
data from Morningstar/yfinance. Capped at 50 funds per run on Streamlit Cloud.
